# Preview Publishing Area

此資料夾只接受 CC 審核通過、可公開且不含私人資料的靜態預覽結果。

## 規則

1. `latest/` 放目前最新版本，供固定手機網址使用。
2. 歷史版本使用 `YYYY-MM-DD_version/` 命名，不覆蓋原始驗收證據。
3. 每份結果至少包含 `index.html` 與其引用的本地圖片／CSS。
4. 不可引用 Future Capsule 正式 API、資料庫或登入狀態。
5. 不可放帳號、電話、Email、訊息內容、設定代碼、Token 或 Cookie。
6. 不可把 Future Capsule 正式原始碼完整複製到此公開儲存庫。
