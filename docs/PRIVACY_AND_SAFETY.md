# 隱私與安全規則

此儲存庫是公開 GitHub Pages 測試通道。任何推送內容都必須視為可被公開讀取。

## 絕對禁止

- 正式帳號、密碼、Cookie、Session、Token 或 API Key。
- 真實姓名、電話、Email、頭像、私人訊息或設定代碼。
- Future Capsule 正式資料庫匯出。
- 含私人資料的手機截圖。
- Future Capsule 未授權公開的完整正式原始碼。
- 可執行送出、刪除、付款或變更正式資料的測試流程。

## 允許內容

- 固定假資料。
- 經人工去識別化的 UI 截圖。
- UI Position Lab 產生的純靜態 HTML 比較總覽。
- 不含商業機密的裝置尺寸與測試說明。

## 發現誤傳時

立即停止後續發布，通知 Sean，並從目前版本及 Git 歷史清除敏感資料；僅刪除最新檔案不足以撤回已提交的秘密資訊。相關 Token 必須立刻撤銷並重新建立。
